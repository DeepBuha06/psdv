import math as m

def mean(data):
    total = sum(data)
    count = len(data)
    mean = total / count
    return mean

def median(data):
    sorted_data = sorted(data)
    count = len(sorted_data)
    
    if count % 2 == 0:
        median = (sorted_data[count // 2 - 1] + sorted_data[count // 2]) / 2
    else:
        median = sorted_data[count // 2]
    
    return median

def std(data):
    mean_value = mean(data)
    squared_diff = [(x - mean_value) ** 2 for x in data]
    variance = sum(squared_diff) / len(data)
    sd = m.sqrt(variance)
    return sd

Immutable data types: an immutable data type is one where, once created, its state cannot be modified. Any attempt to alter its content results in a new object being created instead of modifying the existing one. For example, strings in Python are immutable. Once a string is formed, its characters cannot be changed directly. If you try to assign a new value to a specific index, like s[3] = 'i', a TypeError will be raised because item assignment is not allowed.Another common example is a tuple.

mutable data types: an mutable data type is one where , we can make changes after it is created . like in list we can add , remove and change the state of any index of the list. we can make l[2]=2 directly on list l. set and dictionaries are also mutable.

import matplotlib.pyplot as plt

def fibo(n):
    l=[0,1]
    a,b=0,1
    for _ in range(n-2):
        a,b=b,a+b
        l.append(b)
    return l
